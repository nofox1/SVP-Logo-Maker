# SVP-Logo-Maker
## Description
So the purpose of this project is to help anyone one with making a random but simple logo. Could be for a small business or even for personal use. This super simple project will help many get a basic logo up and going.

## Installation 
Fork and clone to local machine. Then follow these steps:
```
npm i
node index.js
```
you will then be prompted with a series of basic questions.
